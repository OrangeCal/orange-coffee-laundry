# Orange Coffee & Laundry

Sitio web principal para Orange Coffee & Laundry.

## Cómo subir a GitHub

```bash
git clone https://github.com/OrangeCal/orange-coffee-laundry.git
cd orange-coffee-laundry
# Copia aquí todos los archivos de este proyecto
git add .
git commit -m "Primera versión del sitio web"
git push origin main
```

## Cómo correr localmente

```bash
npm install
npm run dev
```

Después abre: http://localhost:3000

## Publicar en Vercel

1. Entra a Vercel.
2. New Project.
3. Importa `OrangeCal/orange-coffee-laundry`.
4. Deploy.
